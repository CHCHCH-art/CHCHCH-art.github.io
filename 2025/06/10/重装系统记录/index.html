重装系统记录

发表于2025-06-10|更新于2025-06-10
|浏览量:

Windows 8.1/8.1 pro 激活密匙：

334NH-RXG76-64THK-C7CKG-D3VPT、XHQ8N-C3MCJ-RQXB6-WCHYG-C9WKB

制作装机 U 盘：

Windows 10 可以直接使用装机程序，Win 11 应当也有相应程序，只需要直接执行程序即可

其余版本需要首先找到 IOS 镜像，然后用 Refus 制作装机 U 盘

PS：U 盘不支持热插拔！！必须要等到数据不再写入、读取的时候软件层面弹出后，再进行物理拔出

重装系统：

首先开机后连续按热键进入 BIOS 界面，设置引导设备顺序

后面正常就能进入装机界面，可能会需要输入密匙，可以参考网上对应版本、配置的激活密匙

自定义安装或者保留配置安装，自己按需要选择

自定义安装中，需要清空系统盘、恢复分区等一系列分区，然后在该闲置区域上重装系统

老台式机重装系统遇到过的问题：

1.卡在复制 Windows 文件 0% 不动

补充：切换为其他版本的 Windows 后成功跳过该状态

推测：U 盘制作不成功、某些版本的 Windows 可能装机文件存在一定问题

注：Win 8 会在这一步报错格式化失败，Win 10 只会卡死不动，所以也可能是格式化失败的问题

注：这一步应该包括，创建分区、格式化分区，相当于新建、格式化

2.成功安装 Windows 并进入 Pin 登录界面登录后，黑屏，无法正常显示

补充：任何按键都无法产生效果，无法跳出管理器、命令行等，初始配置时多次出现需要重试的情况

推测：硬盘损坏、显卡驱动问题

3.自定义安装时任何分区都无法格式化，报错：0x80070057

补充：这部分在先前是不存在的，在成功装 Windows 后实际上一直在循环修复磁盘，反复重启修复

后续在删除对应分区后似乎就逐渐出现这个问题

推测：硬盘损坏
